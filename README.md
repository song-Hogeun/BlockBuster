Block Buster
===========

### 블럭을 부수며 기록을 갱신해보세요!

> 게임 플레이 방법   

플레이 시작 시 블록들이 서서히 내려옵니다.
화면을 스와이프하여 점프할 방향을 조준하고, 놓으면 점프합니다.
블럭을 부수면 한 번의 점프 횟수를 얻습니다.
특수 블록들(J, G, L)이 존재합니다.   
<br>

#### J 블록
![JumpBlock](Assets/06.%20Images/JumpBlock.png)   
-----------
일정 시간동안 제한 없이 점프할 수 있습니다.

#### G 블록
![GroundBlock](Assets/06.%20Images/GroundBlock.png)  
-----------
잠시 쉬어갈 수 있는 땅이 생성됩니다.
일정 시간이 지나면 사라집니다.

#### L 블록
![LifeBlock](Assets/06.%20Images/LifeBlock.png)  
-----------
보너스 목숨를 획득합니다.


### 게임을 진행할 수록 블럭이 빠르게 내려옵니다!
